Para correr el programa primero compila el archivo Expresiones.g4 usando el comando antlr4 Expresiones.g4

una vez se generen los archivos compila todos los archivos java usando el comando javac *.java

una vez hecho esto inicia el programa usando el comando java Calculadora

luego eliga la opcion que desee, se puede ingresar los datos por consola o se puede ingresar un txt donde esten todos las operaciones a realizar
